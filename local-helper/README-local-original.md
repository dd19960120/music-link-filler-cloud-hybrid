# 多平台搜索回填链接工具（汽水接口版）

本工具是本地网页服务。输入歌曲名、歌手名、歌词片段、专辑名或词曲作者等关键词后，可批量搜索并返回歌曲名、歌手名、专辑名、词作者、曲作者和歌曲链接。

## 使用方式

1. 先完整解压整个文件夹。
2. Windows 双击 `start-windows.bat`。
3. 浏览器打开 `http://localhost:5173`。
4. 勾选需要的平台，输入关键词，一行一个。
5. 点击“搜索并回填”。

## 汽水音乐

- 汽水音乐使用本机 PC 客户端数据目录里的登录态和设备参数。
- 工具会自动读取：
  - `%APPDATA%\SodaMusic\DeviceV1`
  - `%APPDATA%\SodaMusic\Network\Cookies`
- 不会控制鼠标，不会切换汽水窗口，不会影响用户继续操作电脑。
- 如果没有安装或登录汽水 PC 客户端，汽水搜索可能失败。
- 汽水自动读取 Cookie 需要 Node.js 22 或更高版本。
- 汽水结果会额外解析公开分享页，返回点赞、评论、转发数据。

## 统计数据

- QQ 音乐：拿到 songDetail 链接或 songmid 后，可补收藏、在听、评论。
- 网易云音乐：可补红心收藏数、评论总数。
- 酷狗音乐：可补评论总数，使用手机端一致的合并评论数。
- 酷我音乐：可补评论总数。
- 酷狗、酷我的手机客户端收藏量/在听人数目前没有从公开网页接口确认到稳定字段，工具不会伪造这些数据。

## 注意

- 不要直接双击 `public/index.html`，必须先启动本地 Node 服务。
- 如 5173 端口被占用，可在命令行使用：

```powershell
$env:PORT="5174"; node server.js
```

- QQ 音乐仍保留 Cookie 模式和专用浏览器模式。
- QQ 专用浏览器模式会滚动/翻页读取更多结果，不再只读首屏 10 条。
- 汽水音乐会按 `next_cursor` 分页读取，数量按页面里的“每个平台候选数”执行。
